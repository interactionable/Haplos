Button
This is an 'impulse'-type button. When you tap it, the library sends out a single OSC message.
Button
This is a 'canvas'-type button. When you press it, the library sends out a single OSC message. When you release the button, another message is sent.
Button
This is an 'node'-type button. When you press it, the library sends out a single OSC message. With your finger on the button, you can move it around. As you do, OSC messages are sent indicating where your finger is using an (x, y) coordinate system. When you finally release the button, another message is sent.
